# Object-Oriented Programming

<!-- kb:status Draft -->

> A programming paradigm based on the concept of objects that contain data and code.

## 一句话理解

OOP organizes code around objects (data + behavior) rather than functions, using encapsulation, inheritance, and polymorphism.
