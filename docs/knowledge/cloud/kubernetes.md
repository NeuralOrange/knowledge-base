# Kubernetes

<!-- kb:status Draft -->

> An open-source container orchestration platform that automates deployment,
> scaling, and management of containerized applications.

## 一句话理解

Kubernetes is the operating system for the cloud — it decides where containers
run, how they talk to each other, and what to do when they fail.
