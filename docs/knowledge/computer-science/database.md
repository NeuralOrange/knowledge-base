# Database

<!-- kb:status Draft -->

> An organized collection of structured data, typically stored and accessed electronically.

## 一句话理解

Databases store, organize, and retrieve data — they are the memory of applications.

## 为什么会出现

File-based storage couldn't handle concurrent access, data integrity, or complex queries. Databases provide ACID guarantees and declarative querying.

## 解决什么问题

- Persistent data storage
- Concurrent access with consistency
- Complex queries and indexing
- Data integrity and relationships
