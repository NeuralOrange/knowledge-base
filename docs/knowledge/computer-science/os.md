# Operating System

<!-- kb:status Draft -->

> The software that manages hardware resources and provides services for application programs.

## 一句话理解

The OS is the bridge between hardware and applications — managing CPU, memory, I/O, and processes.

## 为什么会出现

As computers became more complex, running programs directly on hardware became impractical. The OS abstracts hardware complexity.

## 解决什么问题

- Process scheduling and management
- Memory allocation and protection
- File system abstraction
- Device I/O management
