# Network

<!-- kb:status Draft -->

> The interconnection of computing devices for data communication and resource sharing.

## 一句话理解

Networks enable communication between computers — from local LANs to the global internet.

## 为什么会出现

The need to share resources and communicate between isolated computers drove the development of networking protocols.

## 解决什么问题

- Data transmission between systems
- Resource sharing (files, printers, compute)
- Distributed system communication
