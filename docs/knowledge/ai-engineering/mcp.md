# MCP Protocol (Model Context Protocol)

<!-- kb:status Draft -->

> An open protocol that standardizes how AI models interact with external tools
> and data sources.

## 一句话理解

MCP is like a USB-C for AI — a standard interface for connecting models to tools,
databases, APIs, and file systems.
