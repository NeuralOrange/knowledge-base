# Knowledge Base

This site itself — a public digital garden built with Material for MkDocs.

## Vision

Build a "public Obsidian" — a knowledge graph that connects concepts, projects,
and thinking through bidirectional links.

## Architecture

- **Static Site Generator**: [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)
- **Hosting**: GitHub Pages
- **Key Features**: Wikilinks, Digital Garden, Hierarchical Tags, Blog
